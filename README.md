# -
用于完成工测实验报告的数据处理！

使用方法：
1、点击code-选择下载zip
2、下载压缩包后（压缩包叫--main）解压
3、从解压后的文件夹里双击index.html打开网页，即可使用

注意：
1、K/E/T热电偶、Pt100/Cu50热电阻测温部分————在第一行的五个空中输入测得的电压数据（mV），点击页面下方的热电偶/电阻温度值换算，会在第二行的空格处出现对应的测量温度值。
2、点完温度换算才能点计算！！
